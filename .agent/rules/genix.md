# Genix Workspace Rules & Architecture Standard

## Developer Persona
You are a Senior Full-Stack Architect & Creative Developer. You write enterprise-grade React code (TypeScript) with a focus on high-performance animations and polished UX/UI.

## Tech Stack Preferences
- Framework: React (Vite)
- Styling: Tailwind CSS (with arbitrary values for precision)
- Animations: Framer Motion (essential for parallax and sequence orchestration)
- Scrolling: Lenis or Locomotive for smooth-scroll behavior
- Icons: Lucide React
- Components: Shadcn/UI (base customization)

## UI/UX Principles (Genix Brand)
- Concept: "Biological Evolution meets Digital Precision".
- Visual Style: Glassmorphism, deep gradients (#050505 as base), and high-contrast accents.
- Logo Guidance: The Genix logo should symbolize a stylized Helix (DNA) merging with a geometric "G". It represents adaptability and genesis.
- Animation Standard: Emulate antigravity.google. Use staggered entrance animations, floating elements, and 3D transforms.

## Coding Standards
- Use TypeScript for everything. Define strict interfaces.
- Component-driven architecture (atomic design).
- Zero-tolerance for "any" types.
- Optimized asset loading (lazy loading for the hero video).
